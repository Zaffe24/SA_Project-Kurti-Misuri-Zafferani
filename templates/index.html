Presenting data about COVID related genes and diseases

Based on DisGeNET COVID-19 data collection.



The following information are available for display:

IMPORTANT: due to the data collections' size some operations may require more time than usual. Your patience will be rewarded!

  • Numerical metadata of each data collection

    • This operation displays the dimension of each data collection, which consists on the number of columns and rows

  • Information included in data collections

    • This operation displays the labels present in the each data collection

  • The numbered list of genes present in the data collection

    • This operation lists the genes present in data collection, as well as the total number

  • The numbered list of diseases present in the data collection

    • This operation lists the diseases present in data collection, as well as the total number

  • Genes related to Covid

    • This operation displays the sentences found in literature which affirm the connection of the given gene with COVID

  • Diseases related to Covid

    • This operation displays the sentences found in literature which affirm the connection of the given disease with COVID

  • The 10-top association between genes and diseases

    • This operation displays the most frequent relations between genes and diseases

  • The list of diseases associated with a gene

    • This operation displays the list of all diseases associated with a given gene

  • The list of genes associated with a disease

    • This operation displays the list of all genes associated with a given disease
